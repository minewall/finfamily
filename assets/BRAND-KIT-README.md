# Haile · Logo Kit

Kit de logomarca da Haile, derivado do wordmark original e alinhado à identidade visual (indigo + cream + DM Sans).

## Estrutura
- `Haile-Brand-Kit.pdf` — guidelines de uso (5 páginas)
- `svg/` — wordmark e mark em vetor (indigo, indigo-deep, black, white)
- `png/` — wordmark em PNG transparente e com fundos da paleta
- `favicon/` — mark recortado para app icon, favicon.ico, apple-touch-icon
- `social/` — tiles 1024×1024 para avatar, OG image, splash
- `tokens.css` — variáveis CSS prontas para uso

## Sugestões de uso
- **Logo primário:** `svg/haile-wordmark-indigo.svg` sobre cream/branco
- **Logo reverse:** `svg/haile-wordmark-white.svg` sobre ink ou indigo-deep
- **App icon iOS/Android:** `social/haile-tile-indigo-1024.png`
- **Favicon:** `favicon/favicon.ico`

A logo original foi preservada — o swoosh que conecta o "h" ao "a" é o elemento distintivo e foi mantido em todas as variações.
